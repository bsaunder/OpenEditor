Supported Tags
 
Below are the Tags Implemented in the XHTML 1.1 Tag Set in OpenEditor. Clicking on the tag will take you to the W3C Usage Reference for that Tag.

 

Basic Tags  
<!DOCTYPE>  Defines the document type
<html> Defines a html document
<body> Defines the body element
<h1> to <h6> Defines header 1 to header 6
<p> Defines a paragraph
<br /> Inserts a single line break
<hr /> Defines a horizontal rule
<!--...--> Defines a comment
   
Char Format  
<b> Defines bold text
<font> Defines the font face, size, and color of text
<i> Defines italic text
<em> Defines emphasized text 
<big> Defines big text
<strong> Defines strong text
<small> Defines small text
<sup> Defines superscripted text
<sub> Defines subscripted text
<u> Defines underlined text
   
Output  
<pre> Defines preformatted text
<code> Defines computer code text
<tt> Defines teletype text
<kbd> Defines keyboard text
<dfn> Defines a definition term
<var> Defines a variable
<samp> Defines sample computer code
   
Blocks  
<acronym> Defines an acronym
<abbr> Defines an abbreviation
<address> Defines an address element
<blockquote> Defines an long quotation
<center> Defines centered text
<q> Defines a short quotation
<cite> Defines a citation
   
Links  
<a> Defines an anchor
<link> Defines a resource reference
   
Frames  
<frame> Defines a sub window (a frame)
<frameset> Defines a set of frames
<noframes> Defines a noframe section
   
Input  
<form> Defines a form 
<input /> Defines an input field
<textarea> Defines a text area
<button> Defines a push button
<select> Defines a selectable list
<optgroup> Defines an option group
<option> Defines an item in a list box
   
Lists  
<ul> Defines an unordered list
<ol> Defines an ordered list
<li> Defines a list item
<dl> Defines a definition list
<dt> Defines a definition term
<dd> Defines a definition description
   
Images  
<img /> Defines an image
<map> Defines an image map 
<area /> Defines an area inside an image map
   
Tables  
<table> Defines a table
<caption> Defines a table caption
<th> Defines a table header
<tr> Defines a table row
<td> Defines a table cell
   
Styles  
<style> Defines a style definition
<div> Defines a section in a document
<span> Defines a section in a document
   
Meta Info  
<head> Defines information about the document
<title> Defines the document title
<meta> Defines meta information
   
Programming  
<script> Defines a script
<noscript> Defines a noscript section
